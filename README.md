genicon
====

Generate nice icon.

## Usage

```
$ bundle install --path vendor/bundle
$ bundle exec rake g
$ ls output
sample.png
```

## Dependency

on Mac OS X

```
$ brew install pkg-config imagemagick ghostscript
```
